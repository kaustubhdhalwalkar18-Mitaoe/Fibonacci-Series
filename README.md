# Fibonacci-Series
🔢 Fibonacci Series Generator in Python

This project is a Python program that generates the Fibonacci series up to a given number of terms provided by the user.

The Fibonacci sequence is a mathematical series where each number is the sum of the two preceding numbers, starting from 0 and 1.

Sequence example:
0, 1, 1, 2, 3, 5, 8, 13, 21, ...
